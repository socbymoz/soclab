Step 2: Alerts

When logs trigger a notification
What is an Alert?

A log is a record. An alert is a log that matches a malicious pattern — it's a notification saying "look at this." Not all logs are alerts, but all alerts are logs.

Info = Normal event Warning = Worth checking High = Likely malicious Critical = Attack in progress
Alert Examples from This Lab
[CRITICAL] C2 Beacon — 10.10.10.50 -> 185.220.101.1:9050 (Tor)
[CRITICAL] Audit Log Cleared — backup_admin cleared Security log
[CRITICAL] Data Exfiltration — 10.10.10.50 -> 34.253.159.159:80
[HIGH]     Registry Modified — Persistence key added
[HIGH]     New Admin Account — backup_admin added to Administrators
[WARNING]  Brute Force — 12 failed logins from 45.33.32.156
Rule of thumb: Critical = act now. High = investigate today. Warning = monitor. Info = ignore.
