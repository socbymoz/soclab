Step 8: Improve

Learn and prevent the next incident
What is the Improve Phase?

Improve is the final phase — identify gaps, implement lessons learned, and strengthen defenses. Every incident should make you stronger. If you don't improve, you'll get compromised the same way again.

Lessons Learned from This Breach
GAP FOUND                    → REMEDIATION
──────────────────────────────────────────────
SSH password auth enabled    → Key-only + MFA
No failed login alerting     → Threshold alert (5/min)
No network segmentation      → VLANs for DMZ/Prod
No EDR on servers            → Deploy EDR to all servers
Weak password policy         → 16-char minimum + complexity
No IR playbook               → Create and test playbook

PRIORITY ACTIONS (Next 30 Days):
[Week 1] Deploy EDR to all servers
[Week 1] Disable SSH password auth
[Week 2] Implement network segmentation
[Week 2] Deploy MFA for all admins
[Week 4] Create and test IR playbook
Key mindset: The goal isn't just to fix this incident — it's to prevent the next one. Every gap you close makes your organization harder to breach.
