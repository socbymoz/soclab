Module 11: Malware Concepts for SOC
Sub-Domains / Rooms
  • Malware Analysis Fundamentals — Static analysis (hashes, strings, metadata), dynamic analysis (sandbox execution), and YARA rule creation.
  • Malware Detection in SOC Context — How to identify malware infections from SIEM alerts, EDR telemetry, and network traffic patterns.
Example Analysis: A suspicious file "invoice.exe" is flagged by EDR. The analyst performs static analysis: checks the SHA256 hash on VirusTotal — 15/70 detections as "Trojan.Generic". Runs strings and finds URLs to evil.com/beacon and registry keys for persistence. The PE file is unsigned and compiled with MinGW (unusual for a legitimate document). Dynamic analysis in a sandbox shows the malware creates a scheduled task, modifies the Windows Registry Run key, and establishes C2 via HTTPS to 45.33.32.156.
