Module 1: Blue Team Introduction
Sub-Domains / Rooms
  • Junior Security Analyst Intro — Learn the responsibilities of a junior SOC analyst: monitoring dashboards, triaging alerts, and following runbooks.
  • SOC Role in Blue Team — Understand how the Blue Team defends organizations, the difference between blue and red teams, and SOC structure.
  • Humans as Attack Vectors — Social engineering tactics: phishing, pretexting, baiting, tailgating, and how human error leads to breaches.
  • Systems as Attack Vectors — Unpatched software, misconfigured services, default credentials, and vulnerable network services.
Example Investigation: A junior SOC analyst sees an alert: "Multiple failed logins from external IP 203.0.113.5 to RDP port 3389." The analyst checks the SIEM, finds 500 failed attempts (Event ID 4625) in 10 minutes targeting the admin account. This is a system attack vector — exposed RDP with weak credentials. The analyst isolates the server and escalates to L2.
Key Skills Learned
  • Understanding SOC tiers and analyst responsibilities
  • Recognizing human and system attack vectors
  • Basic alert triage and escalation procedures
  • CIA triad and security fundamentals
