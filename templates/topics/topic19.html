Elaborated Notes with Examples
1. CTI Types — Strategic / Tactical / Operational / Technical

Strategic CTI: high-level intelligence for executives (threat landscape, geopolitical risks). Tactical CTI: TTPs, attack patterns, targeting intelligence — used by defenders to improve controls. Operational CTI: specific campaigns, threat actor profiles — used by SOC leads for hunting. Technical CTI: IOCs (hashes, IPs, domains, YARA rules) — used by analysts and tools for detection.

Example (Strategic): Quarterly report to CISO: "Ransomware-as-a-Service groups (LockBit, BlackCat, Clop) have increased targeting of healthcare sector by 40% in Q2 2026. Our healthcare subsidiary should implement enhanced email filtering and offline backups."
Example (Tactical): Threat intel report: "TA577 group uses ISO file attachments with LNK shortcuts to download BumbleBee malware via MSHTA. Implement detection: Sysmon Event 1 for mshta.exe spawning from explorer.exe."
Example (Technical): IOC feed update: IP:185.130.5.201, Domain:evil-c2.ddns.net, Hash:SHA256:a1b2c3d4e5f6..., Rule:YARA rule for JupiterLoader. These feed directly into SIEM blocklists and EDR detections.
2. OSINT

Open Source Intelligence (OSINT) gathers information from publicly available sources: WHOIS/RDAP, Shodan, Censys, VirusTotal, AlienVault OTX, Twitter/Telegram (threat actor channels), and Pastebin. OSINT is used to track attackers, identify infrastructure, and gather context on campaigns.

Example OSINT Investigation: Analyst investigates IP 185.130.5.201:
- Shodan: Port 443 open serving a self-signed certificate with CN *.evil-c2.ddns.net
- WHOIS: Domain evil-c2.ddns.net registered 3 days ago via Njalla, registrant email temp@mailinator.com
- Censys: TLS certificate fingerprint matches JA3 signature for Cobalt Strike
- Pastebin: A public paste contains the same IP and domain with comment "testing TA577 beacons"
- Twitter Search: Tweet from @malware_tracker mentions "New C2: 185.130.5.201:443 associated with TA577"
- AlienVault OTX: Pulse "TA577_Campaign_2026" includes this IP as an IOC
3. IOC Lifecycle

IOCs have a finite lifespan from identification to obsolescence. Phases: (1) Discovery — identified during incident or threat intel, (2) Validation — confirmed as malicious, (3) Enrichment — context added (actor, campaign, TTPs), (4) Dissemination — shared via MISP, ThreatConnect, SIEM feeds, (5) Active Monitoring — used in detection rules, (6) Expiration/Retirement — IOC becomes stale (IP reassigned, domain expired, or too common).

Example IOC Lifecycle:
Day 1: Analyst discovers IP 185.130.5.201 during incident response. Status: New
Day 1: VirusTotal confirms 15 detections, AbuseIPDB shows 47 reports. Status: Confirmed
Day 2: Analyst tags with TA441, CobaltStrike, adds to MISP event #142. Status: Enriched
Day 2: IP added to firewall blocklist and SIEM watchlist via automated feed. Status: Active
Day 60: IP no longer resolves, no new reports in 30 days. Status: Expiring
Day 90: Removed from active blocklist, moved to historical database. Status: Retired
4. MISP / VirusTotal

MISP (Malware Information Sharing Platform) is an open-source threat intel platform for sharing IOCs and TTPs. It supports structured data with correlations between events. VirusTotal is a file/URL/IP analysis platform. Integration: MISP automatically queries VT for hash lookups, and VT feeds can be imported into MISP as events.

Example MISP Workflow:
(1) Incident responder creates a MISP event: Event #142 — TA441 Cobalt Strike Activity June 2026
(2) Adds attributes: domain:evil-c2.ddns.net, ip-dst:185.130.5.201, filename:invoice.docm, md5:a1b2c3d4e5f6..., yara:rule JupiterLoader {...}
(3) MISP's correlation engine links this domain to 3 other events (same domain, different IPs) — suggesting the same actor.
(4) Correlation rule in Splunk: | lookup misp_ioc.csv ip OR domain OUTPUT description | search description=* — every alert containing these IOCs is tagged with the MISP event reference.
(5) MISP automatically pushes IOCs to firewall, SIEM, and EDR via export modules (Suricata rules, CSV, JSON).
(6) VirusTotal lookup on the hash from the MISP event shows 42/70 detections and "Community" tab confirms it's Cobalt Strike beacon packed with UPX.
