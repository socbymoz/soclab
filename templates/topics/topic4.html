Module 4: Cyber Defence Frameworks
Sub-Domains / Rooms
  • Pyramid of Pain — A model showing how difficult it is for attackers to change different types of IOCs: hashes (easy) → IPs → domains → tools → TTPs (hard).
  • Cyber Kill Chain — 7 phases of an attack: Reconnaissance, Weaponization, Delivery, Exploitation, Installation, C2, Actions on Objectives.
  • Unified Kill Chain — Combines multiple kill chain models into a comprehensive framework covering the entire attack lifecycle.
  • MITRE ATT&CK — Industry-standard knowledge base of adversary tactics and techniques mapped to real-world attacks.
  • Summit — Apply frameworks to analyze a real-world breach scenario.
  • Eviction — Practice evicting an attacker by applying defensive frameworks.
Example: A SOC analyst maps a ransomware alert to MITRE ATT&CK: T1486 (Data Encrypted for Impact). Then traces back through the Cyber Kill Chain: initial access via T1566 (Phishing) → execution via T1059 (PowerShell) → persistence via T1547 (Registry Run Keys). The Pyramid of Pain shows that blocking the hash is easy, but the attacker can change it. Blocking the TTP (PowerShell abuse) is harder and more effective.
Key Skills Learned

  • Understanding kill chain phases for detection
  • Using Pyramid of Pain for defense strategy
  • Analyzing attacks through multiple frameworks
