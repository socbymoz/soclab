Module 3: Core SOC Solutions
Sub-Domains / Rooms
  • Introduction to EDR — Endpoint Detection & Response: agents that monitor endpoints for malicious activity, isolate hosts, and provide forensic data.
  • Introduction to SIEM — Security Information & Event Management: log aggregation, correlation, alerting, and compliance reporting.
  • Splunk: The Basics — Search Processing Language (SPL), indexers, forwarders, dashboards, and alert creation.
  • Elastic Stack: The Basics — Elasticsearch, Logstash, Kibana (ELK): data ingestion, visualization, and detection rules.
  • Introduction to SOAR — Security Orchestration, Automation, and Response: playbook automation, case management, and threat intel integration.
Example: An EDR agent detects a suspicious process (powershell.exe -enc ...) running on an endpoint. The alert is sent to the SIEM, which correlates it with other telemetry. The SIEM triggers a correlation rule: "Suspicious PowerShell + outbound connection to unknown IP". A SOAR playbook automatically enriches the IP via VirusTotal, checks if the endpoint is critical, and opens a ticket in the case management system.

  • Elasticsearch/Kibana fundamentals
