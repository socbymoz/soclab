Module 9: Windows Security Monitoring
Sub-Domains / Rooms
  • Windows Logging for SOC — Understand Windows Event Log channels (Security, System, Application), Event IDs, and log collection with Sysmon.
  • Windows Threat Detection 1 — Detect common threats: brute force attacks, account misuse, and privilege escalation via Event ID analysis.
  • Windows Threat Detection 2 — Advanced detection: process injection, DLL hijacking, and persistence mechanism identification.
  • Windows Threat Detection 3 — Investigate ransomware, credential dumping (Mimikatz), and lateral movement using Windows event logs.
Example Detection: An EDR alert shows WINWORD.EXE spawning cmd.exe. The analyst checks Sysmon Event ID 1 (process creation):
Parent: WINWORD.EXE, Child: cmd.exe /c powershell -enc SQBFAFgAIAAoAE4AZQB...
Sysmon Event ID 3 shows a network connection to 45.33.32.156:4444. Event ID 11 (file creation) shows a .ps1 file written to %TEMP%. The analyst also finds Event ID 13 (registry modification) adding a Run key for persistence. This is a classic phishing → macro malware → C2 attack chain on Windows.
