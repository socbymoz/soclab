Example Snort Rule:
alert tcp $HOME_NET any -> $EXTERNAL_NET 443 (
  msg:"Possible C2 Beaconing";
  flow:to_server;
  detection_filter: track by_src, count 30, seconds 60;
  sid:1000001; rev:1;)
This rule alerts when any internal host makes 30+ connections to port 443 within 60 seconds — indicative of C2 beaconing. The analyst would then investigate the destination IPs, check threat intel, and examine the endpoint for malware.
