Module 12: Threat Analysis Tools
Sub-Domains / Rooms
  • Intro to Cyber Threat Intel — CTI types (strategic, tactical, operational, technical), threat feeds, and intelligence lifecycle.
  • File and Hash Threat Intel — Use VirusTotal, MalwareBazaar, and Hybrid Analysis for file and hash reputation checks.
  • IP and Domain Threat Intel — Check IP and domain reputation via AbuseIPDB, VirusTotal, URLScan.io, and passive DNS.
  • Invite Only — Advanced threat intelligence tools used in professional SOC environments (requires invite).
Example: An analyst sees an alert with IP 45.33.32.156. They check VirusTotal: the IP is associated with 12 malware samples and 3 C2 domains. AbuseIPDB shows 45 reports of SSH brute force and port scanning. URLScan.io shows the IP hosted a phishing page targeting Microsoft credentials. Passive DNS reveals the IP hosted 15 other domains, all registered 2 days ago. The analyst blocks all 15 domains and the IP at the firewall.
