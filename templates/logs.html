Step 1: Logs

Where every investigation begins
What is a Log?

A log is a time-stamped record of an event on a system, app, or network device. Logs are the raw evidence — like a black box on a plane. Without them, you can't know what happened.

Event ID 4624 = Login success Event ID 4625 = Login failed Event ID 1102 = Log cleared
Real Log Examples
Windows Security Log:
LogName: Security   EventID: 4625   User: CORP\unknown
Account Name: admin   Source IP: 192.168.1.50   Status: bad password

Web Server Log (Apache):
192.168.1.50 - - [15/Jun/2026:08:00:12] "POST /wp-login.php" 302 345

Firewall Log (Syslog):
Jun 15 09:00:00 fw-01 %ASA-6-302013: TCP outbound 10.10.10.50:49157 -> 185.220.101.1:9050

SSH Auth Log:

Sysmon Log:
EventID: 1   Process: cmd.exe   User: backup_admin
CommandLine: net user hacker Password123! /add
